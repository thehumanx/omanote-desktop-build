# Notes

Notes are for anything you want to write down and keep — a thought, a meeting summary, a draft, a list. They're written in **markdown**, so light formatting just works.

## Writing

Start a note from the canvas composer (plain text becomes a note) or from the Notes view. The editor supports markdown: headings, **bold**, *italic*, lists, quotes, and links. A few keys to know:

- **Enter** — new paragraph
- **Shift + Enter** — a line break within a paragraph
- **Cmd/Ctrl + Enter** — save

Paste a link into a note and it's kept as a clickable link; notes with a link also show a preview card.

## Folders

Group notes into folders with custom icons, the same way todos and bookmarks work. Uncategorized notes stay together until you file them.

## Hashtags

Tag notes with `#` to connect them to related todos, events, and other notes. Tagged items show up together in the Explore map and in search.

## Trash and restore

Deleting a note moves it to the trash rather than removing it immediately, so you can restore it if you change your mind. Each artifact type has its own trashed view.

## Sharing

A note folder can be shared as a public link — see **Public links**.
